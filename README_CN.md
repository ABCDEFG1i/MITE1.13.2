# MITE1.13.2
**_一个人不可能只是搞了几个胡萝卜然后制成夜视药水，去到地底世界挖钻石就一夜暴富_**
## 特性
* 让你更难采集资源
* 怪物更具有攻击性
* 续航性更差
* 添加多种怪物、食物、工具、护甲
## 如何安装
* 点击上方的 release 选择一个版本，下载MITE1.13.2.jar
* 使用任意启动器安装原版1.13.2
* 将下载下来的MITE1.13.2.jar重命名为1.13.2.jar
* 替换原版的1.13.2.jar
* 启动游戏
